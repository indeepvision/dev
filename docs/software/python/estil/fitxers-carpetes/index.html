Fitxers i carpetes

Sobre l'estructura de carpetes d'un projecte de TypeScript i els noms dels fitxers.

Arxius de Python​

Els arxius de Python S'HAN s'escriuen sempre en snake_case, per exemple action_analysis.py. Si no, no funcionen els imports.

danger

En Python ni els fitxer ni les carpetes poden tenir guions (-), només guions baixos (_).

Noms de carpetes​

Fer servir snake_case (amb guions baixos) per als noms de les carpetes. Per exemple: inference_server, data_models, etc.

Capçalera d'arxius​

En tots els arxius copiar la següent capçalera:

"""
STRICTLY CONFIDENTIAL
Copyright 2025 ID Vision Software SL. All Rights Reserved.
This file is part of a software which is released under a commercial license.

Author/s:
    T. Races
"""
note

Antigament teníem la descripció de l'arxiu a la capçalera, però l'hem eliminat perquè no aportava res. Si passeu per algun arxiu on hi hagi el camp Description: (a sota de l'autor) el podeu eliminar.
