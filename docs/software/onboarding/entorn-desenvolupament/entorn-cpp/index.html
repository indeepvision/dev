Entorn C++

Atenció

Només cal fer aquests passos en cas que hagis de desenvolupar i compilar en C++.

Atenció

No cal instal·lar l'entorn de Linux a no ser que estiguis desenvolupant vision-elements o brain-server.

En aquest apartat s'explica com instal·lar les dependències per desenvolupament amb C++.

  • Veure aquest document per la instal·lació de dependències de C++ per Windows.

  • Veure aquest document per la instal·lació de dependències de C++ per Linux.

Dependències​

En aquest apartat només presentem les dependències i les seves versions. Per les instruccions d'instal·lació, veure els documents específics per Windows i Linux.

Important

Mantenir aquestes versions actualitzades.

CMake 3.31.7​

Pot funcionar amb versions més antigues, però aquesta és la versió testejada tant per Linux com per Windows.

Ninja 1.10.0​

Segurament pot funcionar amb altres versions, no s'ha testat.

Compilador C++​

Windows: Build tools MSVC v141 i MSVC v142 (instal·lar com a components individuals amb Visual Studio Installer).

Linux: GCC 9 i G++ 9.

OpenCV 4.10.0​

Precompilat, copiar dels companys.

Boost 1.71.0​

Precompilat, copiar dels companys.

CUDA 11.8​

Només si has de compilar brain-server.

CUDNN 8.9.6​

Només si has de compilar brain-server.

Tensorflow 2.10​

Només si has de compilar brain-server.

Precompilat, copiar dels companys.

Dependències específiques de Linux​

WSL amb Ubuntu 20.04​

Utilitzar imatge pre-instal·lada de WSL amb Ubuntu 20.04 amb totes les dependències, copiar dels companys. Aquesta imatge ja té instal·lades les versions de les dependències indicades a l'apartat anterior.

Extensions de VS Code per WSL​

S'haurien d'instal·lar automàticament amb les extensions recomanades.
